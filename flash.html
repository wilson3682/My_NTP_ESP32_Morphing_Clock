NTP ESP32 Morphing Clock ver: 1.0

This will install the firmware for the NTP ESP32 HUB75 Morphing Clock

This Clock setup uses two 64x32 HUB75 P4 or P5 Panels.

Once installed, it will create a WiFi access point called:

SSID: NTPMorphingClock

pass: 123456789

Connect to NTPMorphingClock Access Point and enter the password, then connect it to your personal WiFi

Once connected, access the Webpage with the new ip address and change the settings accordingly

NOTE: Make sure to close anything using your devices com port (e.g. Serial monitor)

Your browser does not support the Web Serial API.
